#include <iostream>

using namespace std;

int main(int argc, char ** argv) {
    cout << "Hello " << argv[1] << "  " << argv[2] << endl;

    return 0;
}


Kode C++ ini digunakan untuk menampilkan pesan “Hello” diikuti dua kata yang dimasukkan saat menjalankan program melalui command line.

Contoh:

./program Andi Budi

Output:

Hello Andi  Budi

Jadi, fungsi utama program ini adalah menerima dua argumen dari command line lalu menampilkannya di layar bersama kata “Hello”.
